# EC2インスタンス作成

マネージメントコンソールからAmazon Linuxのインスタンスを作成する。

## リソース名

| 名称 | リソース | 説明 |
| -------- | -------- | -------- |
| cetc-cli-server    | EC2     | AWS CLIの実行環境     |
| cetc-cli-role    | IAMロール     | EC2インスタンスで稼働するアプリケーションに権限を付与する     |
| cetc-ssh-sg | セキュリティグループ | EC2インスタンスにSSH通信を許可する |
| cetc | キーペア | EC2インスタンスログイン時の認証に使う |
