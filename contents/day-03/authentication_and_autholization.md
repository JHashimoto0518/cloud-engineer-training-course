# 認証と認可

>  一般的 な アカウント 管理 では、 利用 者 ごと に ユーザー アカウント を 作成 し ます。 そして、 ユーザー アカウント と、 パスワード などの 利用 者 本人 のみ が 知っ て いる 情報 を 使っ て、 ユーザー アカウント の 正当性 を 確認 し ます。 こうした 正当性 の 確認 を 認証 と いい ます。
